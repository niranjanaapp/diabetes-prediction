# Diabetes Prediction 🧠💉

## 📌 Objective
Predict whether a patient has diabetes using medical data and machine learning.

## 🛠 Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## 📊 Dataset
Pima Indians Diabetes dataset with features like Glucose, BMI, Age, Blood Pressure.

## 🔍 Workflow
Data Cleaning → EDA → Scaling → Logistic Regression → Hyperparameter Tuning → Evaluation

## 📈 Results
✔ Accuracy ~75%  
✔ Cross-validation applied  
✔ Predictions added to dataset  

## 📉 Visualizations
Heatmap, Boxplot, Scatter Plot, Confusion Matrix, Bar Chart

## ✅ Conclusion
The model successfully predicts diabetes risk and demonstrates a complete ML pipeline.
