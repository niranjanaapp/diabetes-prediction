# diabetes-prediction
This project predicts whether a person has diabetes using Machine Learning. The model is trained on medical data such as glucose level, BMI, age, and insulin , 
Dataset - PIMA Indians Diabetes Dataset , 
Tools Used - Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Google Colab , 
Model - Logistic Regression , 
Steps - Data cleaning, Remove spaces, Missing value check, Visualization, Train-test split, Model training, Prediction, Evaluation , 
Output - Prediction (Diabetes / No Diabetes), Accuracy score, Graphs .
